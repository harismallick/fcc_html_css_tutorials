Text can also be display in BOLD or in Italics using their specific tags.
You can also underline your text or use the emphasis tags, which by default converts to italics.
The STRONG tag does the same thing as the tag for bold.

Hyperlinking text on the webpage:
To achieve this, you have to use the <a> and </a> tags.
Anchor tags are used with various different arguments. Some of these include:
- href="url" - This argument allows you to add the url of the webpage the text should forward you to.
- target="" - This argument allows you to pass whether you want the url to be opened on the same tab or a different browser tab.
- To open on a new browser tab, use the "_blank" option.

Lists in html

- Two types of lists can be created in html: ordered and unordered lists.
- These can be created using the <ol> + </ol> and <ul> + </ul> tags.
- Each item in the list needs to placed inside the list tags <li> + </li>
- Here's an example of an unordered list of fruits:
  • apples
    • gala
    • pink lady
    • jazz
  • mangoes
  • oranges
  • pineapples
  • grapes
- Here's an example of an ordered list of fruits:
  1. apples
  2. mangoes
  3. oranges
  4. pineapples
  5. grapes
- You can even create nested lists by placing another list and list elements tag inside a list element.

Tables in html

As an example, create a montly budget table for a household:
Month Rent Utilities Groceries Dining out Entertainment
August $1500 $150 $350 $100 $50
This table looks very basic. It can be made prettier and easier to read by using table tag attributes.
Month Rent Utilities Groceries Dining out Entertainment
August $1500 $150 $350 $100 $50
You can also split columns and rows using colspan and rowspan attributes.
Season Month Rent Utilities Groceries Dining out Entertainment
Autumn August $1500 $100 $50 $350 $100 $50
September $1500 $100 $50 $350 $100 $50
October $1500 $100 $50 $350 $100 $50
