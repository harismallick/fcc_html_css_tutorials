Adding abbreviations to html:
This is achieved using the <abbr> in-line element. You can write the abbreviation, and when the end-user hovers over the abbreviated text, they will be able to see the full version. For example:
GB
USA
Disclaimer: this feature may not be available on all screen readers. So don't abbreviate words that are essential to understanding the text on the page.

There are three types of lists that can be created in HTML:

  • Ordered lists - The different elements are separated by numbers.
  • Unordered lists - The different elements are separated by bullets.
  • Description lists - There are instantiated with <dl> tag.
    These lists contain elements under two tags: the description term (instantiated with <dt>) and the description (created with <dd>)
